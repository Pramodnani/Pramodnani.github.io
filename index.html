Twitter Sentiment Analysis for Covid-19

I developed a Twitter sentiment Dashboard for Covid19 tweets in Kansas using the Dash python framework and deployed it using NGINX server.

  • The data (tweets with the keyword Covid-19) is collected using a Python script which makes use of the GetOldTweets3 library.
  • The collection runs everyday at 12.01 AM and slices off the current day's tweets.
  • Bash scripts are controlled by crontab schedulers to execute the data collection process.
  • Tech Stack used: Python, Dash, Dash routing, Plotly, TextBlob
